# 龙棋传说 - 自走棋游戏策划案 (v2.0)

## 一、项目概述

### 1.1 游戏名称
**龙棋传说** (Dragon Chess Legends)

### 1.2 游戏类型
- **核心玩法**：Roguelike自走棋（杀戮尖塔/小丑牌风格）
- **视觉风格**：D&D龙与地下城奇幻风格
- **题材融合**：中国象棋棋子 + 西幻RPG元素

### 1.3 核心概念
- 7大类棋子，每个大类下有多种变体
- 单人爬塔玩法，每层随机敌人
- 构建自己的棋组，挑战越来越强的对手
- 走法大体符合中国象棋，规则简化

## 三、单人 Roguelike 玩法

### 3.1 爬塔流程
```
[开始] → [第一章：新手森林] → [第二章：废弃矿井] → [第三章：冰霜洞穴]
   ↓         ↓                    ↓                    ↓
[选棋子]   [Boss战]            [Boss战]            [最终Boss]
              ↓                    ↓                    ↓
           [奖励]               [奖励]               [通关/结束]
              ↓                    ↓
           [下一章]             [下一章]
```

### 3.2 每次战斗
- **对手**：AI控制的棋子（难度递增）
- **我方**：玩家编组的棋子（最多上场X个）
- **胜利**：敌方全灭
- **失败**：扣除生命值，生命0则结束

### 3.3 战利品
- 战斗胜利后选择：
  - 🐴 **新棋子**（随机3选1）
  - 💰 **金币**（用于商店）
  - 📦 **装备**
  - 💊 **恢复生命**

### 3.4 商店系统
- 花费金币购买棋子/装备
- 刷新商品
- 出售不要的棋子

### 3.5 升级/合成
- 3个同类型棋子 → 合成1个2星
- 3个同类型2星 → 合成1个3星
- 星级提升：属性全面增强

## 四、属性系统

### 4.1 基础属性
| 属性 | 说明 |
|------|------|
| 生命值 (HP) | 生命归零则死亡 |
| 攻击力 (ATK) | 每次攻击的伤害 |
| 护甲 (DEF) | 减免物理伤害 |
| 魔抗 (MR) | 减免魔法伤害 |
| 速度 (SPD) | 决定行动顺序 |
| 暴击率 (CRIT) | 暴击概率 |
| 暴击伤害 (CDMG) | 暴击时伤害倍率 |

### 4.2 特殊属性
| 属性 | 说明 |
|------|------|
| 闪避 | 概率完全躲避攻击 |
| 吸血 | 攻击回复生命 |
| 护盾 | 额外生命值 |
| 减伤 | 百分比减少伤害 |
| 增伤 | 百分比增加伤害 |

## 六、战斗系统

### 6.1 回合制
- 双方按速度属性依次行动
- 每单位每回合可：移动 → 攻击（或只做其中之一）
- 行动顺序：速度 > 随机

### 6.2 AI敌人设计
- 每关敌人有不同的"棋组"
- 难度通过敌人星级、数量、羁绊提升
- Boss关：强大敌人 + 特殊机制

## 十、总结

**核心体验**：
- Roguelike爬塔玩法，每次开局不同
- 中国象棋棋子为基底，职业/元素变体为深度
- 策略构筑 + 运气成分（小丑牌元素）
- 简单上手，深度策略

**差异化**：
- 纯中式象棋题材 + 西幻包装
- 相比传统自走棋：单人可控性更强，roguelike更有重复游玩价值
